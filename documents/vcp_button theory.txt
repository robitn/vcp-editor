
1. Button Creation Flow
Add a button (Toolbar Button) to Grid
    - In the Inspector Button tab, "New Button" button. (NEW)
        - Button Editor Modal: Top-Left text-input and button: "Enter button name: |________| [OK]"
        - the dialog expands (animates) to reveal remainder of dialog, 
            - XML preview auto-populates with <vcp_button>... (more to follow)
            - if SVG exists, render it; otherwise "Image Missing" placeholder

2. Default Image Selection
    - Browse starts in button's folder
    - If file is outside button folder, copy it in and optionally rename to {buttonName}.svg
    - If previous file was outside button folder, it's likely other are outside as well.

3. Skin Events List
    Snippet from PLC source code:
        ;Maximum Number of Skin Events is 255

        SkinSpinOverPlus_M            IS SV_SKIN_EVENT_1  ; Row  1 Column 1
        SkinSpinAutoMan_M             IS SV_SKIN_EVENT_2  ; Row  1 Column 2
        SkinAux1_M                    IS SV_SKIN_EVENT_3  ; Row  1 Column 3
        SkinAux2_M                    IS SV_SKIN_EVENT_4  ; Row  1 Column 4

    Ignore trailing comments; the trailing _M is an internal symbol
