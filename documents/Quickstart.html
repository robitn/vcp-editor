VCP Editor — Quickstart

One-page quick reference

Overview

Design VCP layouts by placing borders, images, and buttons on a grid. This quickstart covers opening files, adding elements, editing, saving, and troubleshooting.

Open / Create

  • Open: File → Open or Cmd+O.
  • New: File → New — choose a save location.

Add Elements

Use the toolbar to add Border, Image, or Button. Click a grid cell to place the element.

Select & Edit

  • Click an element to select it. The Inspector shows properties.
  • Hover to reveal resize handles and drag to resize.
  • Click border edges to select borders (prevents selecting underlying elements).
  • Right-click for context menu actions: Cut, Copy, Paste, Delete, Add.

Save & Export

  • Save: Cmd+S or File → Save.
  • Export: File → Export to CNC… (packages folder + assets).

Undo / Redo / Clipboard

Undo: Cmd+Z. Redo: Cmd+Shift+Z. Cut/Copy/Paste via toolbar or context menu; Paste is disabled on occupied cells.

Images & Buttons

SVG icons adapt to theme. If an image is replaced on disk, click the toolbar ⟳ Images to refresh the UI.

Troubleshooting

  • Image not showing: click ⟳ Images.
  • Devtools visible in release: ensure `devtools=false` in tauri.conf.json and remove dev menu items.
  • Border selection blocked: click the border edge directly.

Shortcuts

  • Cmd+S — Save
  • Cmd+O — Open
  • Cmd+Z — Undo
  • Cmd+Shift+Z — Redo
  • Delete — Delete selection
